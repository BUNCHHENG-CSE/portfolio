# Hang Bunchheng - Personal Portfolio

A modern, high-performance personal portfolio website built to showcase my experience as a Full Stack Developer and Systems Architect.

## 🚀 About the Project

This portfolio is engineered with a focus on scalable architecture and seamless user experience. It highlights my core competencies in frontend web development, backend API design, and system administration, featuring responsive design and smooth section transitions.

### Built With

*   **Framework:** [Next.js](https://nextjs.org/) (App Router)
*   **Language:** [TypeScript](https://www.typescriptlang.org/)
*   **Styling:** [Tailwind CSS](https://tailwindcss.com/)
*   **UI Components:** [shadcn/ui](https://ui.shadcn.com/)
*   **Package Manager:** [Bun](https://bun.sh/) / npm

## 📂 Project Structure

The project utilizes a modular component architecture for maintainability:

*   `/app`: Contains the Next.js App Router logic, global layouts, and main pages.
*   `/components/features`: Contains isolated UI sections (Hero, About, Experience, Projects, Footer).
*   `/components/ui`: Contains reusable base UI components (Buttons, Cards, etc.).
*   `/public`: Static assets and icons.

## 💻 Getting Started

### Prerequisites

Make sure you have [Node.js](https://nodejs.org/) installed along with your preferred package manager (npm, yarn, pnpm, or bun).

### Installation

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/bunchheng-cse/portfolio.git](https://github.com/bunchheng-cse/portfolio.git)
    ```
2.  **Navigate into the directory:**
    ```bash
    cd portfolio
    ```
3.  **Install dependencies:**
    ```bash
    npm install
    # or bun install
    ```

### Running the Development Server

Start the local development server:

```bash
npm run dev
# or bun dev
```
Open http://localhost:3000 with your browser to see the live site.
